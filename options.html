Antigram

About

Antigram is a simple chrome extension that makes the user able to disable the "Explore" page on Instagram. This way, one can use instagram on the web and see what their friends are up to, without falling into an infinite backhole of meaningless content and procastination. Additionally, the follower suggestions can also be hidden, and the background color can be customized. Luckily, for some reason the Instagram web has no adverts, so that was not needed to block.

Antigram tries to give the user tools to fight back applications designed to be addictive. It is free and it does not track you.
